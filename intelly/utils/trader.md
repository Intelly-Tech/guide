# Trader

{% hint style="info" %}
**Intelly: Trader** interacts with **Intelly: Access** for role management, **Intelly: Estate** (**INFT**) for invest operation and **Intelly: Token** (**INTL**) to receive payments with **Intelly: Oracle** price feeder.
{% endhint %}
